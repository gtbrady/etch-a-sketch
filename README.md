# etch-a-sketch
This is the second project in The Odin Project - Foundations course. It is a browser etch-a-sketch or sketchpad. 

It initializes with a 16X16 grid of divs, using a Flexbox layout for creating the grid. The user is able to draw on the grid by hovering their mouse over the grid.

The user can change the grid detail by hitting the grid detail button. The size of the canvas will remain the same but the size of the individual divs will adjust based on user input (limited to 100x100 grid detail). Hitting the reset button will reset the canvas.

project-planning.txt contains requirements/notes taken while planning for the project.
